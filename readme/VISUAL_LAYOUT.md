## Interaction States

### Video Containers
1. **Default**: Transparent border, subtle shadow
2. **Hover**: (No change - auto-play)
3. **Speaking**: Blue border + glow effect
4. **Disconnected**: Removed from DOM

### Controls
1. **Enabled**: Light gray background
2. **Hover**: Slightly darker gray + scale(1.05)
3. **Active**: Scale(0.95)
4. **Disabled**: Red background (#f44336)

## Grid Behavior

### Participants List
- **Display**: Flex column
- **Overflow**: Auto (scrollable)
- **Gap**: 12px between items
- **Aspect ratio**: 16:9 maintained
- **Scrollbar**: Custom styled (8px, dark theme)

### Empty State
- **Centered**: Text-align center
- **Padding**: 24px vertical, 16px horizontal
- **Font size**: 0.9rem
